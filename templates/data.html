Smoothed Data
Every 15 minutes, the previous 15-minutes worth of raw data values are averaged, stored and then instantly plotted onto the graphs below. This is done to smooth out the graphs and cater for anomalies.

Please select a data field to view its graph:

Predicted Input Data (Hourly)
Every hour, predicted sensor data are calculated and OpenWeatherMap forecast data are obtained using historical sensor data and OpenWeatherMap's hourly-forecast API, respectively. This predicted set of input data is then stored and instantly plotted onto the graphs below.

Please select a data field to view its graph:

Predicted Output Air Pollution (Hourly)
Every hour, the predicted input data from the section above is passed through predictive models to generate air pollution values. Two predicted air pollution values are calculated based on sensor inputs vs OpenWeatherMap API inputs. The two values are then stored and instantly plotted onto the graphs below.
